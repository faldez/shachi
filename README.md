## Features
- Browse image and video posts from booru servers
- Read comments for the post
- Favorite post
- Backup and restore
- Saved searches to see newest posts you care about
- Hide or mute rated posts
- Download post
- and more!

Supported booru engines:
- Danbooru
- Moebooru
- Gelbooru
